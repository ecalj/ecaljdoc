## vitepress メモ

### ローカル
```bash
npm install  <- 初回のみ
npm run docs:dev
```
`npm run docs:dev` ではmarkdown ファイルを更新したら自動的にビルドされる。もしくは

```bash
npm run docs:build
npm run docs:preview
```
