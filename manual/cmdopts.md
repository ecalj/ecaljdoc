# Command-line options for `lmf` / `lmfa` / `lmchk` (and friends)

The Fortran binaries (`lmf`, `lmfa`, `lmchk`, the GW utilities,
the post-processing tools) share one minimal argument parser
in [`SRC/subroutines/m_ext.f90`](https://github.com/tkotani/ecalj/blob/master/SRC/subroutines/m_ext.f90):
`cmdopt0('--foo')` is an exact-match flag,
`cmdopt2('--foo=', outs)` returns the rest after `--foo=` in `outs`.
Any other text is treated as the sname argument (`ctrl.<sname>`).

This page is the catalogue. It is generated by searching
`grep -rE "cmdopt[02]\\('--" SRC/subroutines/ SRC/main/`,
so if a flag goes missing from the source it disappears here
on the next refresh. The companion runtime-override mechanism
(`-v[<toml-path>]=<value>`, e.g. `-v[ham.scaledsigma]=0.8`) is
described in [`toml_migration`](./toml_migration#run-time-v-overrides);
it is **not** a `cmdopt` and is not listed here.

The "Used in" column points at the file that first uses the flag —
follow it for the precise semantics if a row below is too terse.

## How to read the table

- **Flags** are `cmdopt0` (no value): write them verbatim, e.g. `--writeham`.
- **Key=value** are `cmdopt2`: always include `=`, e.g. `--jobgw=1`.
  Don't drop the `=`; the parser does prefix-only matching, so
  `--pr` and `--print=foo` would collide if `=` were optional.
- Most flags work in `lmf` (and `lmf --jobgw=…` which acts as the
  GW driver). `lmfa` / `lmchk` honour the few that apply to the
  atomic-program / structure-check phases — they are explicitly
  marked below.

## Run-time TOML overrides (`-v[<path>]=<value>`)

For overriding TOML keys at run time, use the **bracketed** `-v` form;
the legacy `-v<name>=<value>` (`%const` substitution) has no parser
and is silently ignored. Examples:

```bash
lmf si -v[bz.nkabc]=[8,8,8] -v[ham.scaledsigma]=0.8
lmf si -v[ham.so]=1 -v[ham.nspin]=2     # SOC as perturbation
lmf si -v[spec.1.r]=2.5                  # override SPEC[1].r
```

The path is dotted (`section.key`); `[[spec]]` entries are 1-indexed.
Applied overrides are logged on rank 0. See
[`toml_migration#run-time-v-overrides`](./toml_migration#run-time-v-overrides)
for the full spec.

## When this page goes stale

If the source has a flag this page doesn't list (or vice versa), trust
the source. Refresh with:

```bash
grep -rEoh "cmdopt[02]\('[-][^']+'" \
     SRC/subroutines/ SRC/main/ | sort -u
```

and reconcile by hand. We deliberately do not auto-generate this page
because the categorical grouping and the prose notes are the value;
a flat alphabetical dump is what `grep` already gives you.
